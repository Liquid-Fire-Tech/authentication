## Installation
Add this line to your application's Gemfile:

```ruby
gem 'authentication'
```

And then execute:
```bash
$ bundle
```

Or install it yourself as:
```bash
$ gem install authentication
```

To install
```
$ rails g authentication:install
```

For help
```
$ rails g authentication: --help
```
